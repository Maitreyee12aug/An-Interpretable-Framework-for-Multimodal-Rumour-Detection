# An-Interpretable-Framework-for-Multimodal-Rumour-Detection
Official implementation of "Gated Consistency and  Uncertainty-Aware Fusion: An Interpretable Framework  for Multimodal Rumour Detection" (The Visual Computer,  Springer, 2025). Features CLIP-based GNN, Dynamic Gating,  and MC Dropout fusion. If you use this code, please cite  our manuscript.
